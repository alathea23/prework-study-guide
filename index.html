Prework Study Guide

HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • Each content item (i.e. element) is wrapped in an HTML tag that explains what the item is and how to render it
  • The term 'parent' and 'child' are used to explain the tiering structure within HTML

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • You can find color options on this website ref:https://htmlcolorcodes.com/color-picker/ Insert # then color number
  • You can easily adjust image and font size by section value or self-created categorization for bulk formatting
  • Explore the funtionality and versatility of font/shadow formatting more!

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • -b: creates a new branch
  • git commit: commits changes made in a branch
